== Description ==
Add a META tag Author for SEO reasons. Author name can be automatic on each post/page or set to a choosen name.

== Installation ==

1. Extract the plugin folder from the downloaded ZIP file.
2. Upload BAW Google Author folder to your /wp-content/plugins/ directory.
3. Activate the plugin from the "Plugins" page in your Dashboard.
4. Go to settings !
